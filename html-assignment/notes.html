Tags List

<html>
Description: The root element of an HTML document. It wraps the entire content of the page.
<head>
Description: Contains meta-information about the HTML document, such as its title, links to stylesheets, and metadata.
<body>
Description: The main content of the HTML document. Everything that you want to display on the webpage, such as text, images, and links, is placed inside this tag.
<h1>
Description: Defines a top-level heading. This is typically used for the main title or important headings on the webpage.
<a>
Description: Defines a hyperlink, which allows users to navigate to another webpage or another section within the same page. The URL is specified using the href attribute.
